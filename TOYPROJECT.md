## 토이 프로젝트

### 주소록 프로그램

#### 개요

- C 기본 학습 후에 구조체, 배열, 포인터, 파일IO, 문자열, 검색, 정렬 등 활용하는 묶음 프로젝트
- DB로 전향하기 전에 가장 좋은 예제.
- GUI(Graphic User Interface)는 추후에 가능!

#### 목표

- 연락처 추가/목록/수정/삭제/검색
- 파일 저장/불러오기
- (추가) 정렬/중복체크/csv로 내보내기

#### 데이터 모델
- 이름(name), 전화(phone), 이메일(email), 주소(addr), 메모(memo)
- 구조체로 생성
